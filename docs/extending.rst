Extending Aloe's TestCase
=========================

.. toctree::
    :maxdepth: 2

.. autoclass:: aloe.testclass.TestCase()

   Aloe runs all tests within a :class:`unittest.TestCase`. You can extend
   this class to run your tests with certain other features, i.e. using
   Django's ``TestCase``.
